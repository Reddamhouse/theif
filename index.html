About

Our IST Reddam group aimed to create an interactive film experience that utilises multiple aspects of multimedia. The idea of this 'film' is that a cat burglar is breaking into a house attempting to steal a priceless artifact, throughout the film they user is faced with multiple decisions which, when chosen incorrectly will result in the death of the burglar and the end of the 'film'.

We used electron to create desktop apps based off an HTML5 frontend. The library p5.js is used to programm the options in this film. For the mobile apps we used phonegap, which was very similar to the electron process.
